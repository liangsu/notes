## 代码
用TAB键起始的段落，会被认为是代码块

	public class A{
		public static void main(String[] args){
			System.out.println("hello markdown!");
		}
	}

如果在一个行内需要引用代码，只要用反引号引起来就好，比如这样：  在java中输出一段文字是这样写的`System.out.println("hello markdown!")`，这样既可以输出了
